ABOUT ME

The "full stack developer" is a term that refers to a developer capable of creating a computer tool (software, application, website ...) all alone from A to Z. It thus supports the complete stacking of components of this tool. However, with the current market that is very demanding and the increasing emergence of new technologies, some are beginning to wonder if it is still possible to manage everything alone.

According to Peter Yared, Sapho's chief technology officer, "Full Stack was possible in the Web 2.0 era. But, a new generation of startups has emerged, pushing the boundaries of virtually every area of ​​software. From artificial intelligence to data analysis to connected objects, via the mobile. It becomes virtually impossible for a single developer to program through the full stack of modern technologies. "
